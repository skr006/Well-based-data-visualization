# Well-based-data-visualization
A full-stack application where the user selects specific well and visualizes its time-series measurements.

## Stack
- `React`
- `Flask`

## Instructions
> Note: Please use the following instructions to work with this version.

### Execute Frontend
```
cd frontend
npm run dev
```

### Execute Backend
```
cd Backend
python app.py
```
### Explorative Data Analysis
1. Dataset Information and description
2. Removal of columns with no values
3. Sample well graph for verification
